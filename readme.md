# Controle de Séries

Sistema simples para controle de séries feito php com o framework Laravel. No sistema utilizei as seguintes tecnologias
 

- Laravel
- Bootstrap
- JavaScript
- Autenticação com o Auth
- Envio de email
- Upload de arquivos
